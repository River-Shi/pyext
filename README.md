## Install Codon
Download [Codon](https://github.com/exaloop/codon/releases) and install with
```
/bin/bash -c "$(curl -fsSL https://exaloop.io/install.sh)"
```

## Build
```python
python3 setup.py build_ext --inplace
```

## Install Package
```
pip install codon-jit
```

## Run test
```
python test.py
```
